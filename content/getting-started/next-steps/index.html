Reference Materials

In addition to the Getting Started tutorials, this website also includes reference materials for {{ stache.config.product_name_short }}'s HTML, CSS, and JavaScript framework. Currently, the Components section describes engineering tools such as directives and services. The Components section provides examples and code samples for these {{ stache.config.product_name_short }} building blocks.

We are currently expanding the Components section into a References section that will also describes {{ stache.config.product_name_short }} design patterns and layout elements for additional guidance.

  • The Layout section will describe layout elements to build your application such as cards, forms, pages, headers, repeaters, summaries, tabs, and tiles.
  • The Patterns section will describe the design patterns that you can employ to create a consisstent user experience with {{ stache.config.product_name_short }}. Patterns are key concepts behind the design philosophy and provide guidance for design decisions and interaction patterns such as how to add data, alerts and notifications, how to edit data, error handling, filters, highlights, labels, navigation, how to display data, how to remove data, how to search and filter data, options to select data, styling, toasts, and user assistance.

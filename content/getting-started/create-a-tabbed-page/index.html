The previous tutorial described how to create a record page. Now, lets look at how to create the other common page type in {{ stache.config.product_name_short }}: the tabbed page. While record pages display data for individual records, tabbed pages usually serve as overviews for many records, such as a lists of constituents or gifts.

Tabs Component

{{ stache.config.product_name_short }} uses the UI Bootstrap Tabs module to display tabs and adds more {{ stache.config.product_name_short }}-specific functionality with its own Tabset module. In this example, we have two tabs. Since each tab represents a list of records, the number of records in each list is displayed next to the tab header with the CSS class .bb-tab-header-count. Also, since we want the tab content to go edge-to-edge but want the tabs themselves to have a margin, we need to add the .bb-page-tabs CSS class.

This example shows grid data in each tab. For more information about how to use grids, see the grid directive documentation.

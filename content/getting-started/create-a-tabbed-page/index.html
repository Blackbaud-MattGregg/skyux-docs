In the previous step you learned how to create a record page. In this step you'll learn how to create the other common page type in Sky UX: the tabbed page. While record pages are used to show data for individual records, a tabbed page usually serves as an overview of many records, such as a list of constituents or gifts.

Tabs Component

{{ stache.config.product_name_short }} uses the UI Bootstrap Tabs module to display tabs and adds more {{ stache.config.product_name_short }}-specific functionality with its own Tabset module. In this example we have two tabs. Since each tab represents a list of records, the number of records in each list is displayed next to the tab header with the CSS class .bb-tab-header-count. Also, since we want the tab content to go edge-to-edge but want the tabs themselves to have a margin, we will need to add the .bb-page-tabs CSS class.

This example shows grid data in each tab. For more information on how to use grids, see the grid directive documentation.
